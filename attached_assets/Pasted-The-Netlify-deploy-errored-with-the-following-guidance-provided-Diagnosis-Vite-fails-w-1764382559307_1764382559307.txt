The Netlify deploy errored, with the following guidance provided:

**Diagnosis**  
- Vite fails while loading PostCSS because the plugin defined in `postcss.config.js` cannot be found. See [line 75](#L75-L88), where the stack trace shows `Error: Loading PostCSS Plugin failed: Cannot find module 'autoprefixer'`.

**Solution**  
- Add `autoprefixer` to your project dependencies so the plugin referenced in `postcss.config.js` is available during the build. From the project root run:

  ```bash
  npm install --save-dev autoprefixer
  ```

- Commit the updated `package.json` and lockfile, then push to trigger a new Netlify build.
